# cryp-8

chip-8 emulator

## Installation
First of all [Install Crystal](https://crystal-lang.org/install/) and then build with command
```shell script
shards install && shards build
```

## Usage
Run and enjoy (or not).

```shell script
./bin/cryp-8
```
